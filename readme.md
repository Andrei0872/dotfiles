# dotfiles

## Prerequisites

```bash
sudo apt install luarocks
```

Recommended to have installed:

* _ripgrep_
* _fd_

## Set up Neovim

```bash
git clone git@github.com:Andrei0872/dotfiles.git
```

```bash
# Ensure this folder is created.
mkdir ~/.config/nvim
```

```bash
ln -s -f ~/dotfiles/nvim/* ~/.config/nvim
```
